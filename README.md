# Licensr - Suggest Software license in your Terminal Itself! 

This program allows you to decide and apply software licenses because choosing from 100+ Licenses is hard tbh :c 

# Usage 

To start with the wizard, type this 
```bash
licensr wizard
```

after you got your own license recommendation, to apply that license, use this - 
```bash
licensr apply-license GNU_GPL_v3 
```

A Lotta Devs Confuse of which license to use :/ (me too)
